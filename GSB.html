Projet de 2ème année BTS SIO

Ce site web permet aux visiteurs médicaux de saisir leurs frais de déplacement, de restauration et d'hébergement, afin que ces frais puissent être traités pour un éventuel remboursement. Le site a été développé en utilisant les langages de programmation PHP, HTML et CSS, et utilise une base de données MySQL pour stocker les informations saisies.
